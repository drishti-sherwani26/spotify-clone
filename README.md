# Spotify Clone 🎵

A responsive Spotify Clone built using HTML and CSS.

## Features

- Responsive UI
- Sidebar Navigation
- Music Player UI
- Playlist Cards
- Modern Spotify Design

## Tech Stack

- HTML
- CSS

## Project Purpose

This project was built to strengthen my frontend development skills by recreating Spotify's user interface using only HTML and CSS.
